Greetings!
=========
The Soft9000/PyDAO ("PyDAO 9000") Project allows us to quickly create 
a Python Class from either a GUI, API, textual (CSV, TDF, Enpiped, etc.) 
data file, or by using a Python Dictionary.

Best of all the code generated by PyDAO 9000 is 100% Core Python 3 - no other 
files and / or 'tech is required!

Quick Start
===========

(1) Install Python 3
    Source: http://www.python.org

(2) pip install PyDAO-9000
    Source: https://pypi.org/project/PyDAO-9000/

(3) Once installed most will probably want to use the GUI?

>>> from SqltDAO import main as PyDAO
>>> PyDAO.Main.mainloop()

or simply:

>>> import SqltDAO

(4) Finally, enjoy the video!
    YouTube: https://youtu.be/azCokv4iI8k
